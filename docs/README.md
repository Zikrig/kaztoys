# Дизайн-документы проекта Dara Exchange

Набор документов для разработки Telegram-бота обмена игрушками.

| Документ | Содержание |
|----------|------------|
| [01_STRUCTURE.md](01_STRUCTURE.md) | **Структура проекта**: дерево каталогов, слои (handlers, services, models), зависимости, точка входа, переменные окружения. |
| [02_DATABASE.md](02_DATABASE.md) | **База данных**: ER-схема, таблицы (users, listings, responses, matches, subscriptions, search_filters), индексы, генерация кодов. |
| [03_BUSINESS_LOGIC.md](03_BUSINESS_LOGIC.md) | **Бизнес-логика**: FSM (состояния пользователя), правила и инварианты, сервисы, события и уведомления, справочники. |
| [04_USER_STORIES.md](04_USER_STORIES.md) | **Пользовательские истории и экраны**: что видит пользователь на каждом шаге (вход, меню, заявки, поиск, отклики, мэтчи, подписка, поддержка). |

Общий план разработки: [ПЛАН_РАЗРАБОТКИ_БОТА.md](../ПЛАН_РАЗРАБОТКИ_БОТА.md) в корне проекта.

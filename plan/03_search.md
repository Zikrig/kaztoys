# Поиск заявок

Режим поиска: **опрос по фильтрам** → **показ по одной заявке** (не списком из 20).

## 3.1 Первый вход в «Начать поиск»

Опрос из **трёх шагов** (бот запоминает параметры):

- **Шаг 1 — Что ищете?**  
  Кнопки: те же 5 категорий + **«Все категории»**.
- **Шаг 2 — Для какого возраста ищете?**  
  Кнопки: 0–2 | 3–5 | 6–8 | 9–12 | Любой возраст.
- **Шаг 3 — В каком районе?**  
  Кнопки: список районов Астаны + **«Любой район»**.

## 3.2 Повторные входы

- Бот **сразу** показывает заявки по сохранённым фильтрам, без повторного опроса.

## 3.3 Показ одной заявки

- Одна заявка: фото, имя, кол-во сделок, код, **категория, возраст, район**, описание.
- Под заявкой кнопки:
  - **Предложить обмен**
  - **Дальше**
  - **В главное меню**
- В области ввода текста всегда доступна: **«Изменить параметры поиска»** → заново опрос, сохранение новых параметров, показ заявок по ним.

## 3.4 Кнопки в поиске

- **«Предложить обмен»** — флоу отклика (выбрать свою игрушку / создать новую → отправить предложение). После отправки: «Продолжить поиск?» — если да, следующая заявка по тем же фильтрам.
- **«Дальше»** — следующая заявка. Если заявок больше нет: «По вашему запросу больше нет заявок. Пришлём уведомление, когда появится новая!» + кнопки: **Изменить параметры поиска** | **В главное меню**.
- **«В главное меню»** — возврат в меню.

## 3.5 Подсказка после 7 заявок

- Когда пользователь нажал **«Дальше»** 7 раз подряд (без отклика): **одноразовое** сообщение за сессию: «Не нашли нужное? Попробуйте изменить параметры поиска.»  
  Кнопки: **Изменить параметры поиска** | **Продолжить поиск**. После «Продолжить» — листаем дальше, подсказка больше не показывается.

## 3.6 Сортировка

- Заявки в поиске — **по дате**, самые свежие первыми. Без ранжирования и нейросетей.

## 3.7 Уведомления (второй этап)

- Если пользователь долистал до конца (заявок нет) — бот запоминает его фильтры.
- Когда появляется новая заявка, подходящая по категории/возрасту/району — пуш: «Новая игрушка в категории «Машинки», 3–5 лет, район Есиль!» + кнопка **«Посмотреть»**.

Можно реализовать после того, как основной поиск уже работает.
